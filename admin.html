📖 如何获取 GitHub Token？

  1. 登录 github.com
  2. 点击右上角头像 → Settings
  3. 左侧菜单最下方 → Developer settings
  4. 点击 Personal access tokens → Tokens (classic)
  5. 点击 Generate new token (classic)
  6. 勾选 repo 权限，然后点击 Generate token
  7. 复制生成的 token（只显示一次！）
